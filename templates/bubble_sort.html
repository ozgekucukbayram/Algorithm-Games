📘 Bubble Sort – Bubble Dance

You receive 5 shuffled number cards (e.g., 5-2-4-1-3).

On each turn, you may compare and swap only adjacent cards.

Goal: Sort the numbers in ascending order using the fewest swaps possible.

🔍 Algorithm Insight:
This simulates the Bubble Sort algorithm. In the worst case, it runs in O(n²) time.
Each pass "bubbles up" the largest number to the end.

🧠 As the list becomes sorted, no swaps are needed – demonstrating the optimized version of Bubble Sort.

📈 More efficient algorithms for real use cases:
✅ Merge Sort – O(n log n)
✅ Quick Sort – Avg. O(n log n)
